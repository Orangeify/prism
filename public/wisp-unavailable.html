WebSocket backend unavailable on this host

Vercel hosts the static frontend, but it does not support long-lived WebSocket servers. The proxy at /wisp/ is not available here.

To run the full proxy you must host the WebSocket server (the project's index.js) on a provider that supports WebSockets (Fly, Render, Railway, a VPS, etc.).

For deployment instructions and options, see VERCEL.md in the repository.
